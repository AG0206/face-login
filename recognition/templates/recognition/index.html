About Smart Library

The Smart Library is a modern digital library system designed to work across multiple franchise branches. It provides three levels of login (HQ Admin, Franchise Owner, User/Student) and dual authentication methods (Email/Password or Face Recognition).

Franchise Management

Each franchise is managed independently by its owner while being monitored centrally.

Multi-Level Access

Three distinct user roles with appropriate permissions for each level.

Face Recognition

Advanced biometric authentication for secure and convenient access.
